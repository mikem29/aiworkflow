# AI-Assisted Drupal Block Designer Requirements

This folder contains the requirements documentation for the AI assistant that helps Drupal 11 site builders design pages by composing content blocks.

- `product_overview.md` — vision, scope, and user personas.
- `functional_requirements.md` — detailed functional capabilities and user flows.
- `technical_considerations.md` — integration patterns, AI enablement, and non-functional requirements.
