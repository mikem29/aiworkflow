# AI Website Builder Requirements

This folder gathers the requirements documentation for the AI assistant that helps site builders design pages by composing content blocks. The location mirrors the intended product hierarchy: `IndieMade / Features / AI Website builder / requirements`.

- `product_overview.md` — vision, scope, and user personas.
- `functional_requirements.md` — detailed functional capabilities and user flows.
- `technical_considerations.md` — integration patterns, AI enablement, and non-functional requirements.
